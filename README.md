## 📦 包含什么

| Skill | 适用平台 | 功能 |
|-------|---------|------|
| `cn-dev-setup` | Hermes Agent | 一键配置国内开发环境（DeepSeek/通义千问/豆包等API） |
| `cn-coder` | Hermes Agent / Claude Code | 中文代码生成最佳实践（命名规范、注释、文档） |
| `wsl-helper` | Hermes Agent | WSL/Windows 双环境自动化管理 |
| `api-wrapper` | 通用 | 国内API一键封装（DeepSeek/通义千问/百度文心/讯飞星火） |

## 🚀 快速开始

### Hermes Agent

```bash
# 克隆到 skills 目录
git clone https://github.com/lzhao8956-glitch/awesome-dev-skill-pack.git \
  ~/.hermes/skills/awesome-dev-skill-pack

# 重新启动 Hermes，Skills 自动生效
```

### Claude Code

```bash
# 导入 Skill 配置
claude import-skill https://github.com/lzhao8956-glitch/awesome-dev-skill-pack
```
